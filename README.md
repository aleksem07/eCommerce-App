# WEBPACK

Для установки пакетов используйте команду npm install

## Команды

### Запуск сервера для разработки (порт 3000)

```shell
npm run start
```

### Сборка проекта без оптимизации
```shell
npm run build-dev
```

### Сборка проекта с оптимизацией
```shell
npm run build-prod
```

### Очистка папки dist
//unix (linux, macOS)
```shell
npm run clear-u
```
//windows
```shell
npm run clear-w

```

### Запуск линтера

```shell
npm run lint
```

### Запуск линтера (исправление ошибок)
```shell
npm run lint-fix

```
